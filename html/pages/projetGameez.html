Gameez

Plateforme dédiée à la recherche et notation de jeux de société.

Durée: 2 mois

Contexte du Projet

Fan de jeux de société, j'ai voulu créer une plateforme qui recense un maximum de jeux pour permettre aux joueurs de découvrir, noter et d'ajouter des jeux dans leur wishlist. Ce projet a été commencé en parallèle de ma formation DWWM et va être alimenté et amélioré prochainement.

Fonctionnalités Principales

  • Création de comptes utilisateurs
  • Authentification sécurisée
  • Recherche de jeux avec filtre dynamique
  • Ajout de jeux dans une wishlist
  • Consultation et rédaction d'avis (note et commentaire)
  • Gestion des jeux par l'administrateur
  • Gestion des utilisateurs par l'administrateur
  • Forum de discussion entre utilisateurs
  • Stockage de données de manière sécurisée
  • Accessibilité grâce au responsive et à l’ergonomie.

Ce que j'ai appris

  • Organisation et gestion de projet : Utilisation d'outil comme Trello ou Notion.
  • Design : Maquettage, choix de la charte graphique, responsive, ergonomie.
  • Gestion des utilisateurs : Création de compte, gestion sécurisé des informations personnelles, gestion des accès.
  • Gestion base de données : SQL pour les jeux, NoSQL pour le forum. Manipulation de jointure pour les lien entre les jeux et leurs extensions.
  • Gestion filtre dynamique : Les résultats des jeux se met à jour en temps réél.
  • Clareté du code : Structure MVC pour faciliter la clareté et donc la maintenabilité du code.
  • Sécurité du code : Utilisation de requêtes préparées, hash de mot de passe, token CSRF, sécurisation des input pour contrer les failles connues. Utilisation d'outils dédié à la sécurité comme SonarQube.

Technologies utilisées

HTML5 CSS3 Javascript PHP 8.3 MySQL MongoDB Docker Heroku
