Ecoride - Plateforme de covoiturage en France

Application dédiée aux personnes à la recherche de covoiturage, soucieuse de l'environnement

Durée: 3 mois | Rôle: Développeur |

Contexte du Projet

J'ai créé Ecoride dans le cadre de ma formation Développeur Web & Web Mobile pour permettre aux personnes soucieuse de l'environnement de trouver des covoiturages en France principalement avec des véhicules électriques. Ce projet a été réalisé en parallèle de ma montée en compétences.

Fonctionnalités Principales

  • Création de comptes utilisateurs
  • Authentification sécurisée
  • Publication et gestion de trajet par les « conducteurs »
  • Recherche et réservation de trajet par les « passagers »
  • Utilisation de crédits pour la réservation de trajet
  • Gestion des utilisateurs, trajets, réservations par l’administrateur et employé
  • Gestion des employés par l’administrateur
  • Stockage de données de manière sécurisée
  • Accessibilité grâce au responsive et à l’ergonomie.

Ce que j'ai appris

  • Comprendre un cahier des charges : Répondre aux besoins clients exprimés à travers les fonctionnalités métier demandées.
  • Organisation et gestion de projet : Utilisation d'outil comme Trello ou Notion.
  • Design : Proposer des maquettes au client puis réaliser l'interface responsive et accessible correspondante.
  • Gestion des utilisateurs : Création de compte avec choix de rôle, gestion sécurisé des informations personnelles, gestion des accès.
  • Utilisation API : Utilisation d'une API pour visualiser le trajet sur une carte.
  • Clareté du code : Structure MVC pour faciliter la clareté et donc la maintenabilité du code
  • Sécurité du code : Utilisation de requêtes préparées, hash de mot de passe, token CSRF, sécurisation des input pour contrer les failles connues. Utilisation d'outil comme SonarQube.

Technologies utilisées

HTML5 CSS3 Tailwind Javascript PHP 8.3 MySQL MongoDB
